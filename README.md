# Led Sequence
## 匯入音檔
- 在音軌那邊按 `+ Import` 的按鈕
## 編輯 sequence
- default 會 align bpm
- 右上角的四個按鈕： `add`, `select`, `delete`, `snap`
    - `add`: 添加一段led，添加的形式取決於你選擇 `color`, `blink`...
    - `select`: 選一段led，在軌裡面可以拉框框做範圍匡選
    - `delete`: 按一下可以刪掉 led
    - `snap`: on 的時候 `add` 和拖移都會 align bpm，off 的時候可以自由移動
## 改顏色
- 按 `` ` `` -> 在 `Palette` 選 `custom`

## 匯出
- 右上角 `file` 選 `Export sequence .zip`
